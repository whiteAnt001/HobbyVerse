## 📝 소개
취미를 같이 즐기고 싶은 사람들이 모임을 구할 수 있는 모임 플랫폼 입니다.

사용자는 키워드 검색, 카테고리 탭에서 원하는 카테고리별로 분류된 모임을 확인할 수 있으며 원하는 모임이 없을 시 모임을 개설할 수 있습니다.
모임 소개와 일정을 확인하고 참가를 할 수 있으며, 참가 신청을 한 뒤에도 참가를 원하지 않으면 참가 취소를 할 수 있습니다.
게시판을 이용해서 다른 사용자들과 소통을 즐길 수 있으며, 중복 가능한 닉네임을 사용하므로 익명성을 보장할 수 있습니다.
문의사항을 통해서 플랫폼 관리자에게 의견을 제시할 수도 있고, 버그 신고 등을 할 수 있습니다.



다음과 같은 내용을 작성할 수 있습니다.
- 프로젝트 소개
- 프로젝트 화면 구성 또는 프로토 타입
- 프로젝트 API 설계
- 사용한 기술 스택
- 프로젝트 아키텍쳐
- 기술적 이슈와 해결 과정
- 프로젝트 팀원

필요한 기술 스택에 대한 logo는 [skills 폴더](/skills/)에서 다운로드 받을 수 있습니다.
